# Code of Conduct

This project adopts the Contributor Covenant v2.1 as its Code of Conduct.

The canonical text is published at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.

Until the full text is committed inline, contributors are expected to follow the principles described at that URL.

**Maintainer note:** to inline the full text, run:

```sh
curl -sSL https://www.contributor-covenant.org/version/2/1/code_of_conduct.md > CODE_OF_CONDUCT.md
```

Reports of unacceptable behavior should be directed via the [`SECURITY.md`](./SECURITY.md) private reporting channel.
